# 🎨 デザイン範例集

這個資料夾包含 3 種不同風格的首頁設計範例。
每個檔案都是獨立的，可以直接在瀏覽器開啟預覽！

---

## 📁 檔案列表

| 檔案 | 風格 | 特色 |
|------|------|------|
| `v1-kawaii-pop.html` | 🌈 繽紛可愛風 | 漸層背景、浮動泡泡、彩虹文字 |
| `v2-minimal-elegant.html` | ✨ 極簡優雅風 | 黑白金配色、大量留白、serif 字體 |
| `v3-retro-gaming.html` | 🎮 復古遊戲風 | 像素字體、霓虹燈效果、掃描線 |

---

## 🖥️ 如何預覽

1. 點擊任一 `.html` 檔案
2. 點擊「Raw」按鈕
3. 另存新檔，然後用瀏覽器開啟

或是直接 clone 下來後開啟檔案！

---

## 💡 使用方式

如果喜歡某個設計：
1. 把該檔案的 `<style>` 區塊內容複製到 `style.css`
2. 參考 HTML 結構調整 `index.html`
3. 可以混搭不同版本的元素！

---

*這些範例不會影響現有的 index.html，放心探索！*
