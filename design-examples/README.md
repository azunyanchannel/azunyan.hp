## 🖥️ 如何預覽

**方法 1：GitHub Pages**
1. 在 GitHub 點擊檔案
2. 複製 Raw 連結
3. 貼到 https://htmlpreview.github.io/? 後面

**方法 2：本地預覽**
1. Clone 這個 repo
2. 直接用瀏覽器開啟 .html 檔案

---

## 💡 使用方式

如果喜歡某個設計：
1. 把該檔案的 `<style>` 區塊內容參考到 `style.css`
2. 參考 HTML 結構調整 `index.html`
3. 可以混搭不同版本的元素！
